# Portafolio Profesional | Nicolle Camila Piñeros Castañeda


> **"Construyendo software con mentalidad de eficiencia industrial."**

Bienvenido a mi repositorio del portafolio profesional. Este proyecto ha sido desarrollado con un enfoque minimalista, moderno y de alto rendimiento.

---

##  Características Clave

* **Diseño Web Vanilla:** Desarrollado utilizando únicamente código nativo (HTML5, CSS3 y JavaScript), evitando el uso de frameworks pesados o librerías externas para garantizar un rendimiento óptimo y una carga ultra veloz.
* **Enfoque Mobile-First & Responsivo:** Adaptabilidad total en dispositivos de escritorio, tablets y móviles, con encuadres de imagen optimizados para mantener el foco visual.
* **Experiencia Cinematográfica:** Integración de un modal interactivo nativo para la reproducción de video en formato horizontal original con fondo oscuro optimizado para celulares.
* **Diseño de Interfaz Premium:** Paleta de colores cuidada, tipografías personalizadas integradas y un sistema de espaciado que prioriza la legibilidad y la fluidez visual.

## 📂 Estructura del Proyecto

El proyecto mantiene una arquitectura de archivos limpia y modular:

```text
PORTAFOLIONICOLLEPINEROS/
├── assets/                  # Imágenes de proyectos, capturas y recursos multimedia
├── fuentes/                 # Tipografías personalizadas del sistema
│   ├── Creato/              # Fuente tipográfica para logotipos y énfasis
│   └── Eternalo/            # Fuente tipográfica premium para títulos principales
└── src/                     # Código fuente de la aplicación
    ├── index.html           # Estructura semántica principal
    ├── script.js            # Lógica interactiva y control del modal de video
    ├── style.css            # Estilos generales y media queries responsivas
    └── variables.css        # Configuración del sistema de diseño (Custom Properties CSS)
